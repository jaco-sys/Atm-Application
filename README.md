# ATM Simulation using Python

## Project Overview

This project is a simple **ATM (Automated Teller Machine) simulation** developed using Python. The program runs in the command-line interface and allows users to perform basic banking operations such as checking account balance, depositing money, withdrawing money, and exiting the system.

The purpose of this project is to demonstrate fundamental Python programming concepts including functions, loops, conditional statements, and user input handling.

## Features

* Check current account balance
* Deposit money into the account
* Withdraw money from the account
* Exit the ATM system safely
* Continuous menu system using loops

## Technologies Used

* Python

## How the Program Works

1. The program starts with a predefined account balance.
2. A menu is displayed with available banking operations.
3. The user selects an option by entering a number.
4. Based on the selected option, the program performs the corresponding operation:

   * Display current balance
   * Deposit money and update balance
   * Withdraw money if sufficient funds are available
   * Exit the program
